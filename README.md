# Student Enrollment Form using JsonPowerDB

## Project Overview

This project is a **Student Enrollment Form** developed using **HTML, CSS, JavaScript, Bootstrap, jQuery, and JsonPowerDB (JPDB)**.

The application allows users to store, retrieve, and update student records in JsonPowerDB through REST APIs.

---

## Database Details

* **Database Name:** SCHOOL-DB
* **Relation Name:** STUDENT-TABLE
* **Primary Key:** Roll-No

---

## Input Fields

* Roll-No
* Full Name
* Class
* Birth Date
* Address
* Enrollment Date

---

## Features

* Save new student records
* Search student by Roll Number
* Update existing records
* Reset form
* Input validation
* REST API integration with JsonPowerDB
* Responsive user interface using Bootstrap

## How to Run

1. Clone or download this repository.
2. Open the project in any code editor.
3. Add your JsonPowerDB Connection Token in `script.js`.
4. Run `index.html` in your browser.
5. Use the form to save and update student records.
